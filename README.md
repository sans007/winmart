# create a New environment

install virtualenv

```
pip install virtualenv
```

create environment

```
 python -m venv .venv

 #or

 virtualenv .venv
```

## activate the env

#### On Windows, run:

```
.venv/Scripts/activate

#or

source .venv/Scripts/activate
```

#### On Unix or MacOS, run:

```
source .venv/bin/activate
```

## deactivate

```
    deactivate
```
